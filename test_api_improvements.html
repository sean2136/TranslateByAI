🤖 问题2：新增Gemini API支持

支持的API提供商：

DeepSeek Google Gemini

🔧 新增功能：

  • API提供商选择：在设置中可选择DeepSeek或Gemini
  • 动态帮助链接：根据选择的提供商显示对应的API Key获取链接
  • 统一API接口：两种API使用相同的调用方式
  • 独立错误处理：针对不同API的专门错误处理
  • API Key验证：支持验证两种API的Key有效性

📋 API获取指南：

  • DeepSeek: platform.deepseek.com
  • Gemini: aistudio.google.com/app/apikey

📍 问题3：字幕位置调整优化

❌ 修改前

底部偏移：60-200px

默认值：80px

字幕位置偏上，容易遮挡内容

✅ 修改后

底部偏移：100-300px

默认值：120px

字幕位置更合适，避免遮挡

调整详情：

  • 最小值：60px → 100px（增加40px）
  • 最大值：200px → 300px（增加100px）
  • 默认值：80px → 120px（增加40px）
  • 调整范围：140px → 200px（增加60px可调范围）

🧪 测试步骤

1. 重新加载插件

刷新插件以应用所有改进

2. 测试API Key动态更新

  1. 不配置API Key，直接在YouTube上启用翻译
  2. 观察错误提示
  3. 打开插件设置，配置API Key
  4. 保存设置后，无需重新加载插件
  5. 再次尝试翻译，应该正常工作

3. 测试Gemini API

  1. 在插件设置中选择"Google Gemini"
  2. 观察帮助链接变化为Google AI Studio
  3. 输入Gemini API Key
  4. 点击"测试翻译"验证
  5. 在YouTube上测试Gemini翻译效果

4. 测试字幕位置

  1. 在插件设置中调整"底部偏移"滑块
  2. 观察范围变为100-300px
  3. 测试不同位置的字幕显示效果
  4. 确认字幕不会遮挡YouTube控制栏
